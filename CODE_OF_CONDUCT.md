# Code of Conduct

All contributors are expected to maintain a respectful and professional environment.

## Expected Behavior

- Communicate respectfully.
- Provide constructive feedback.
- Use public discussions for project-related questions.

## Unacceptable Behavior

- Harassment or abusive language.
- Personal attacks.
- Spamming issues or pull requests.
- Working on assigned issues without permission.

Maintainers may remove content or restrict access if these rules are violated.

By contributing to this project, you agree to follow this Code of Conduct.
